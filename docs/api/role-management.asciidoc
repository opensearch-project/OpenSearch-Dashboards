== {osd} role management APIs

Manage the roles that grant <<opensearch-dashboards-privileges, {osd} privileges>>.

WARNING: Do not use the {ref}/security-api.html#security-role-apis[{opensearch} role management APIs] to manage {osd} roles.

The following {osd} role management APIs are available:

* <<role-management-api-put, Create or update role API>> to create a new {osd} role, or update the attributes of an existing role

* <<role-management-api-get, Get all {osd} roles API>> to retrieve all {osd} roles

* <<role-management-specific-api-get, Get specific role API>> to retrieve a specific role

* <<role-management-api-delete, Delete role API>> to delete a {osd} role
